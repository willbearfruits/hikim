Working surface

A DAW-shaped workbench.

  • Audio and MIDI timeline editing
  • Session grid: launch, scene follow, record takes into slots
  • CAPTURE writes the session jam into the arrangement
  • TEETH effects rack macros and MIDI learn
  • WIRES device and instrument modes, modout to the PATCH bay
  • VST3, AU and LV2 hosting paths
  • Built-in RUST, GRAVEL, HYMN and RUBBLE instruments
  • Patch modulation from LFO, chaos and follower sources
  • ffmpeg bundled: drag in any audio or video format
  • Built-in update check

Source build

CMake, JUCE 8, C++20.

cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)
./build/dawglitch_artefacts/Release/HIKIM

./build/ruin_tests_artefacts/Release/ruin_tests

The test suite covers model, clip, instrument, stretch-cache and TEETH/WIRES passthrough behavior. It is still early software.
